Quartz Monitor API

这是Quartz监控系统的API接口页面。所有功能请通过RESTful API访问。

配置管理

获取所有配置: GET /api/config/list

添加新配置: POST /api/config/add

删除配置: DELETE /api/config/{uuid}

任务管理

获取所有任务: GET /api/job/list

获取任务详情: GET /api/job/{id}

执行任务: POST /api/job/execute

触发器管理

获取所有触发器: GET /api/trigger/list

添加触发器: POST /api/trigger/add

暂停触发器: POST /api/trigger/pause

恢复触发器: POST /api/trigger/resume

删除触发器: DELETE /api/trigger/{id}

Cron表达式

测试Cron表达式: POST /api/cron/test

解析Cron表达式: POST /api/cron/parse

系统信息

获取系统Dashboard: GET /api/dashboard
